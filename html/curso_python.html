Sobre el curso

Objetivo General

Dotar al estudiante de una comprensión profunda de los fundamentos de la programación con Python, permitiéndole no solo escribir código sintácticamente correcto, sino también pensar algorítmicamente. Al finalizar el curso, el estudiante será capaz de desarrollar scripts funcionales, manipular estructuras de datos complejas y aplicar principios de programación modular para la automatización de tareas y el análisis de datos básico.

Lo que aprenderás

El curso se divide en 4 Módulos Estratégicos. Cada módulo sigue el ciclo: Teoría Conceptual -> Demostración Práctica -> Ejercicio Guiado -> Tarea Autónoma.

Contenido del curso

1. Fundamentos del Lenguaje y Lógica Secuencial
2. Control de Flujo y Estructuras de Datos
3. Modularidad y Funciones
4. Persistencia de Datos y Programación Orientada a Objetos (POO)

Sobre el curso

Objetivo General

Capacitar al estudiante para arquitecturar, estructurar y estilizar sitios web funcionales y estéticamente profesionales. Al finalizar el curso, el alumno dominará la creación de interfaces semánticas con HTML5 y diseños responsivos (adaptables a móviles) mediante CSS3, utilizando metodologías de desarrollo modernas como Flexbox y CSS Grid.

Lo que aprenderás

El curso está diseñado bajo una metodología teórico-práctica. No escribiremos código sin entender el "por qué".

  • Herramientas a utilizar: Visual Studio Code (VS Code), Google Chrome (Developer Tools).
  • Metodología: Cada concepto se introduce con teoría breve, se aplica en un ejemplo en vivo (Live Coding) y se refuerza con un ejercicio práctico inmediato.

Contenido del curso

1. Introducción a html 5 min
2. Fundamentos de Estilo y el Modelo de Caja (CSS3) 7 min
3. Flexbox y Grid 12 min
4. Funciones 15 min
5. Manipulación del DOM 20 min

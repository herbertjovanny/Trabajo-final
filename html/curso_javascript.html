Sobre el curso

Introducción

JavaScript ha dejado de ser un simple lenguaje de scripting para convertirse en la columna vertebral de la web moderna. En esta introducción, abordaremos el cambio de paradigma que supuso la llegada de ES6 (ECMAScript 2015) y cómo las actualizaciones anuales han robustecido el lenguaje. Analizaremos por qué el "JavaScript Moderno" prioriza la legibilidad, el manejo seguro de datos y la modularización. Este curso no es solo sobre "cómo escribir código", sino sobre cómo pensar como un ingeniero de software moderno, dejando atrás prácticas obsoletas como el uso de var, los callbacks anidados y la manipulación directa e ineficiente del DOM.

Objetivo General

Dotar al estudiante de una comprensión profunda y práctica del ecosistema actual de JavaScript, trascendiendo la sintaxis básica para dominar las características avanzadas introducidas desde ES6 hasta ES2024. Al finalizar el curso, el alumno será capaz de escribir código limpio, modular y eficiente, manipular datos complejos mediante programación funcional y gestionar la asincronía profesionalmente, preparándolo para el uso de librerías modernas (React, Vue) o desarrollo de backend (Node.js).

Lo que aprenderás

El curso está diseñado bajo una metodología "Learn by Building" (Aprender construyendo). Se divide en 4 Módulos Estratégicos que son incrementales..

  • Nivel de Entrada: Abstracción de sintaxis y nuevas variables.
  • Nivel Lógico: Manipulación de datos y paradigma funcional.
  • Nivel de Entrada: Asincronía y comunicación con servidores.
  • Nivel Arquitectónico: Modularización y organización profesional del código.

Contenido del curso

1. Fundamentos del Nuevo Estándar 5 min
2. Lógica de Datos y Programación Funcional 7 min
3. Asincronía y El Event Loop 12 min
4. Arquitectura, Módulos y Tooling 15 min
